To access project, start XAMPP and run apache and mySQL server

Open htdocs directory (C: -> xampp -> htdocs -> Enterprise Pro)

In browser URL, write path to the file, e.g to view login page:

localhost/enterprisepro/login/login.php



To view student's home page:

localhost/enterprisepro/student/home/studhome.php


UserID, usernames and passwords can be found in MySQL database
